UTMap: Triplet Neural Network for Uncertainty Medical Image Analysis

Research focused on predictive uncertainty, model reliability, and robustness of deep learning classifiers applied to medical computer vision.

Developed and benchmarked a meta-learning framework for uncertainty estimation against traditional baseline methods, demonstrating robust performance across experimental settings.

Interpretable Components Using Genetic Programming Employing Instruction-like Structure

Research focused on interpretable feature extraction using Genetic Programming for dimensionality reduction and classification tasks.

Introduced IGP, a feature extraction method that generates interpretable components through linear and non-linear feature combinations using an instruction-like structure instead of traditional expression trees.
